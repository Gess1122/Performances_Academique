# Performances_Academique
Ce projet propose un dashboard interactif avec Streamlit pour analyser les performances académiques de 500 étudiants. Il inclut le nettoyage des données, la gestion des valeurs manquantes, et des statistiques descriptives pour une exploration facile du dataset.

Le dashboard permet de visualiser la répartition réussite/échec, d’analyser les notes par matière, et d’explorer les profils des étudiants selon l’âge, le genre, la bourse ou l’accès à Internet. Des graphes interactifs et interprétations automatiques mettent en évidence les facteurs influençant la réussite.
